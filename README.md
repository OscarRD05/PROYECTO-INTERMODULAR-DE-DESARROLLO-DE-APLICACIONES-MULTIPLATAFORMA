# APK de la aplicación

## Descarga

ENLACE DEL APK: https://drive.google.com/drive/folders/1HPUPuAFOPNnLZgmDPXAhyO6QOd4TTkjU?usp=sharing

## Instrucciones de instalación

1. Descargar el archivo `.apk` desde el enlace
2. Copiarlo al dispositivo Android (OnePlus 15)
3. Activar "Instalar desde orígenes desconocidos" en Ajustes > Seguridad
4. Abrir el archivo `.apk` desde el móvil para instalarlo

## Requisitos

- Android 5.0 (API 21) o superior
- Móvil con NFC
